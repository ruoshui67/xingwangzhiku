# 2026-06-17

## 技能图标系统（全16张人格图片）
- 将 `人格图片/` 下全部 16 张技能图标复制到 `assets/textures/skill_icons/`
- 命名规则：`{技能英文}_{性别}.png`，如 `withered_rose_m.png`、`rhine_riverbank_f.png`
- `Main.tscn`：用单个 `UI/SkillIcon` TextureRect 替换原来的 `KunRoseIcon` + `KunRoseIconFemale`
- `Npc.gd`：
  - `_SKILL_ICON_MAP` 常量映射技能名→文件名（含 `%s` 占位符用于 m/f 替换）
  - `_update_skill_icon(text)` 提取技能标签，按 `_using_p2` 选男女，`load()` 动态加载贴图
  - `_start_typing` 调用 `_update_skill_icon`
  - 对话关闭/完成时隐藏图标
- 图标位于对话框右侧（offset_left=-200, offset_top=-1120, offset_bottom=-960）

## 终局旁白
- P2 结束莫罗后 → 走近 P1 → 播放终局独白（弗莱 12 句内心独白）
- 独白结束后黑屏 1s → 渐亮 → 杜瓦尔旁白 9 句
- 结尾点"继续" → 黑屏渐入 2s → 永久黑屏结束

## 迪特尔对话修复
- P1 结束迪特尔后禁用 NPC（process_mode=DISABLED）防二次对话
- 独白切换后重新激活迪特尔（P2 可交互）

## NPC 对话存档重构
- 原来只存一个 NPC 的 `_dialogue_index` 和 `_dialogue_finished`
- 改为按名称 `NPC_{name}` 保存每个 NPC 的完整状态：`_dialogue_index`, `_round_index`, `_dialogue_finished`, `_p1/p2_dialogue_ever_completed`, `_is_post_dialogue`, `_using_p2`, `process_mode`, `visible`, 3D位置
- 叙事阶段变量存入 `[NPC_Phase]` 节
- 读档恢复全部状态，可从中断对话继续

## 对话暂停
- ESC 在对话期间暂停游戏并显示暂停覆盖层
- PauseOverlay z_index=100, SettingsPanel z_index=110
- 暂停时递归设置 DialogueBox 子树 mouse_filter=IGNORE 确保按钮可点击
